# ryan.zander

A personal blog site showcasing projects and hobbies.

## Live Site

[ryanzander.vercel.app](https://ryanzander.vercel.app)

## Structure

```
src/
├── index.html          # Homepage with blog list
├── 404.html            # Custom 404 error page
├── styles.css          # Global styles
├── blogs/              # Blog post pages
│   ├── webdevelopment.html
│   ├── sciencefair.html
│   └── bmxbiking.html
└── img/                # Images
    └── removebg.png    # Logo
```

## Features

- Responsive design with mobile support
- Minimal dark theme
- Smooth hover transitions
- Blog posts with dates

## Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- Hosted on Vercel

## Local Development

Open `src/index.html` in a browser, or use a local server:

```bash
npx serve src
```
