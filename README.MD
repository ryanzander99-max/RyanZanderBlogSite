# ryan.zander

A personal blog site showcasing projects and hobbies.

## Live Site

[zander.life](https://zander.life)

## Features

- Responsive design with mobile support
- Minimal dark theme
- Smooth hover transitions
- Blog posts with dates

## Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- Hosted on Vercel

## Local Development

Open `src/index.html` in a browser, or use a local server:

```bash
npx serve src
```
